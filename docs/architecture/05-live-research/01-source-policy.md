# 01. Source Policy

Live Research는 출처 품질이 답변 품질을 좌우하므로 source policy를 명확히 둔다.

## Preferred Sources

우선순위:

1. 공식 기관, 공공 데이터, 신뢰 가능한 식품/영양 기관
2. 원문 레시피 사이트와 공식 브랜드/매체
3. 검색 결과에서 반복 확인되는 다수 출처
4. 블로그/SNS/커뮤니티는 트렌드 신호로만 사용

## Source Metadata

각 evidence는 다음 정보를 가져야 한다.

```json
{
  "title": "문서 제목",
  "url": "https://...",
  "publisher": "출처명",
  "published_at": "2026-05-01",
  "fetched_at": "2026-05-13T12:00:00+09:00",
  "summary": "핵심 요약",
  "confidence": 0.82
}
```

## Citation Rules

- 최신성 기반 답변에는 출처 URL을 함께 제공한다.
- 한 출처만으로 강하게 단정하지 않는다.
- 트렌드성 정보는 “~로 보인다”, “여러 출처에서 언급된다”처럼 표현한다.
- 저작권이 있는 원문 레시피를 길게 복제하지 않는다.

## Excluded Sources

- 출처가 불명확한 복붙 페이지
- 과도한 광고/자동 생성 콘텐츠
- 접근 권한이 필요한 비공개 페이지
- 원문 재사용 금지가 명확한 콘텐츠를 그대로 가져와야 하는 경우
